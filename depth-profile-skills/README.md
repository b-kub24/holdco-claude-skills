# Depth Profile Skills

**4 personality and team analysis skills for Claude AI.**

Designed for managers, coaches, HR professionals, and anyone seeking deeper self-awareness or team optimization.

## Skills Included

| # | Skill | Description |
|---|-------|-------------|
| 1 | **Personality Assessment** | Multi-framework personality profiling (Big Five, DISC, Enneagram, MBTI) |
| 2 | **Communication Style Analysis** | Analyze and improve communication patterns across dimensions |
| 3 | **Team Compatibility Checker** | Evaluate team dynamics, predict friction, optimize collaboration |
| 4 | **Career Path Matcher** | Match personality and skills to optimal career paths |

## Installation

1. Copy the `depth-profile-skills` folder to your Claude skills directory
2. Restart Claude or reload skills
3. Skills trigger automatically based on your prompts

## Quick Start

```
"Give me a full personality assessment. Ask me questions."

"Analyze my communication style based on these emails I'll share."

"I'm building a startup team. Check compatibility between these 3 profiles."

"What career fits my personality? I'm an introvert who loves data and autonomy."
```
